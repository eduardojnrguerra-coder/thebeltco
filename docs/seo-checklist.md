# TheBeltCo SEO Checklist

Use this checklist before publishing any new products, collections, pages, or theme changes to ensure maximum search visibility and Google Rich Results eligibility.

---

## 1. Google Search Console

- [ ] Submit sitemap: `https://yourstore.com/sitemap.xml` in Google Search Console
- [ ] Submit sitemap in Bing Webmaster Tools
- [ ] Verify domain ownership (DNS TXT record or HTML file)
- [ ] Check for crawl errors and fix 404s
- [ ] Review Core Web Vitals report monthly
- [ ] Monitor manual actions and security issues

## 2. Sitemap Submission

Shopify generates `/sitemap.xml` automatically. No action needed on the theme side.
- [ ] Confirm `/sitemap.xml` loads and lists collection/product/page sitemaps
- [ ] Confirm `/robots.txt` (theme-generated) includes `Sitemap: https://yourstore.com/sitemap.xml`

## 3. Product Page Optimisation (per product)

- [ ] **Title** – Unique, descriptive, includes primary keyword (e.g. "Full-Grain Leather Belt – Brown | TheBeltCo")
- [ ] **Description** – Unique (no copy-paste from manufacturer). Include sizing info, leather type, care. 150–300 words.
- [ ] **Image alt text** – Set in Shopify media manager. Describe the product naturally (e.g. "Brown full-grain leather belt with brass buckle on white background"). Do not keyword-stuff.
- [ ] **URL handle** – Short, clean, keyword-rich (e.g. `/products/full-grain-leather-brown-belt`)
- [ ] **Price** – Visible and clearly formatted
- [ ] **SKU** – Unique per variant (helps Google Merchant Center)
- [ ] **Metafields** – Set `custom.leather_type`, `custom.belt_width`, `custom.materials`, `custom.size_guide`, `custom.leather_care` where applicable
- [ ] **Tags** – Use for filtering and badges: `best-seller`, `new`
- [ ] **Vendor** – Set to your brand name consistently
- [ ] **Collections** – Assign each product to at least one collection

### Product Structured Data (auto-generated by theme)

The theme outputs JSON-LD Product schema on every product page. Verify with Google Rich Results Test:
- [ ] Product name is correct
- [ ] Price and currency are present
- [ ] Availability is accurate (InStock/OutOfStock)
- [ ] Image URLs are crawlable
- [ ] SKU is populated
- [ ] Brand is set

## 4. Collection Page Optimisation

- [ ] **Title** – Unique collection title (e.g. "Brown Leather Belts" not "Collection")
- [ ] **Description** – Add 50–200 words of unique content. Do not use the same description on multiple collections.
- [ ] **Image** – Upload a collection image (not just auto-generated)
- [ ] **URL** – Clean handle (e.g. `/collections/brown-leather-belts`)
- [ ] **Products** – At least 1 product, but ideally 4+ for the page to have substance
- [ ] **SEO title** – Set via Shopify SEO settings if default is not optimal

### Filtered/Sorted Pages

Filtered and sorted collection pages (`?sort_by=...`, `?filter.*`) automatically get a `noindex` tag from the theme. This prevents duplicate content index bloat. No action needed.

## 5. Blog & Article Optimisation

- [ ] **Blog title** – Descriptive (e.g. "Leather Care Tips & Style Guides")
- [ ] **Article titles** – Unique, keyword-rich, compelling
- [ ] **Article content** – At least 300 words, original, valuable to readers
- [ ] **Article image** – Add a featured image with descriptive alt text
- [ ] **Article tags** – Use sparingly and consistently (not too many unique tags)
- [ ] **Author name** – Set to a real name for credibility
- [ ] **URL** – Clean handle

## 6. Page Optimisation

- [ ] **Title** – Unique per page
- [ ] **Content** – Original, adds value
- [ ] **Images** – Optimised, with alt text

## 7. Google Merchant Center Setup

- [ ] Create Merchant Center account
- [ ] Verify domain (same as Search Console)
- [ ] Set up shipping settings
- [ ] Set up tax settings
- [ ] Submit product feed (Shopify automatically syncs if the channel is installed)
- [ ] Review product data quality: missing images, missing prices, incorrect availability
- [ ] Fix any disapproved products before running ads

## 8. Technical SEO

- [ ] **Canonical URLs** – Theme automatically sets `<link rel="canonical">` on all pages
- [ ] **`noindex`** – Cart, search results, account pages, and filtered collection pages get `noindex`. Verify.
- [ ] **`robots.txt`** – Review for accuracy (blocking admin, cart, checkout, account, search, tagged pages)
- [ ] **Sitemap** – Confirm submission to Google
- [ ] **PageSpeed** – Run Google PageSpeed Insights on homepage, product page, and collection page
  - Mobile: target 70+
  - Desktop: target 90+
  - Fix: unoptimised images, render-blocking resources, unused CSS/JS
- [ ] **Core Web Vitals** – Monitor in Search Console. LCP < 2.5s, FID < 100ms, CLS < 0.1
- [ ] **Mobile usability** – Test with Google's Mobile-Friendly Test
- [ ] **HTTPS** – Confirm SSL certificate is active (Shopify handles this)
- [ ] **404 handling** – Custom 404 page is in place (theme provides one)
- [ ] **Pagination** – Collection pages use standard pagination. No `prev`/`next` needed (Shopify handles)

## 9. Content Quality

- [ ] No duplicated product descriptions across variants or similar products
- [ ] No thin content (pages with very little text)
- [ ] Internal links between related products, collections, guides
- [ ] Breadcrumb navigation visible on product, collection, article, and page templates

## 10. Redirects (URL Changes)

- [ ] If a product URL changes, set up a 301 redirect in Shopify Admin → Navigation → URL Redirects
- [ ] If a collection URL changes, same process
- [ ] Never delete a product/collection without setting up a redirect if it has existing traffic

## 11. Before Publishing a Theme Update

- [ ] Run `shopify theme check` (Shopify CLI) to validate theme code
- [ ] Test all templates: homepage, product, collection, cart, search, 404, blog, article, page
- [ ] Verify `noindex` is NOT present on product, collection, blog, article, or page templates
- [ ] Verify `noindex` IS present on cart, search, account, filtered collection pages
- [ ] Check that JSON-LD structured data still renders on product pages
- [ ] Check Google Rich Results Test on a sample product page
- [ ] Run Lighthouse audit on homepage and product page
- [ ] Test mobile view for all templates
- [ ] Confirm theme editor still loads all sections

## 12. Ongoing Monitoring

- [ ] Weekly: Check Search Console for new crawl errors or indexing issues
- [ ] Monthly: Run PageSpeed Insights for key pages
- [ ] Monthly: Review product data quality in Merchant Center
- [ ] Quarterly: Full content audit – update stale blog posts, improve thin product descriptions
- [ ] Quarterly: Check competitor rankings and adjust content strategy

## Quick Reference: Theme-Generated SEO Features

| Feature | Status | Location |
|---------|--------|----------|
| `<title>` | ✓ Auto-generated | `layout/theme.liquid` (falls back to theme settings) |
| `<meta description>` | ✓ Auto-generated | `layout/theme.liquid` (falls back to theme settings) |
| `<link rel="canonical">` | ✓ All pages | `layout/theme.liquid` |
| `<meta name="robots" content="noindex">` | ✓ Cart, search, account, filtered collection | `layout/theme.liquid` |
| JSON-LD Product schema | ✓ Product pages | `snippets/structured-data-product.liquid` |
| JSON-LD Organization schema | ✓ All pages | `snippets/structured-data-organization.liquid` |
| JSON-LD BreadcrumbList | ✓ Product, collection, article, page | `snippets/structured-data-breadcrumbs.liquid` |
| JSON-LD Article schema | ✓ Article pages | `layout/theme.liquid` (inline) |
| JSON-LD FAQPage schema | ✓ FAQ section (homepage) | `sections/faq.liquid` |
| JSON-LD CollectionPage | ✓ Collection pages | `layout/theme.liquid` (inline) |
| JSON-LD Blog schema | ✓ Blog pages | `layout/theme.liquid` (inline) |
| JSON-LD WebSite schema | ✓ Homepage | `layout/theme.liquid` (inline) |
| JSON-LD WebPage schema | ✓ Page template | `layout/theme.liquid` (inline) |
| Open Graph tags | ✓ All pages | `snippets/social-meta-tags.liquid` |
| Twitter Cards | ✓ All pages | `snippets/social-meta-tags.liquid` |
| `<h1>` | ✓ One per page | Each main template section |
| Semantic HTML | ✓ `<nav>`, `<main>`, `<article>`, `<footer>`, etc. | All templates |
| Image alt text | ✓ From Shopify media alt text | All image tags |
| Breadcrumb navigation | ✓ Product, collection, blog, article, page, cart, search | `snippets/breadcrumb.liquid` |
| `robots.txt` | ✓ Custom | `robots.txt.liquid` |
| Sitemap link in robots.txt | ✓ | `robots.txt.liquid` |
| Internal links to collections | ✓ In footer | `sections/footer.liquid` |
